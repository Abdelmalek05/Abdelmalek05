## 🚀 Featured Projects

### 🛡️ Pixel — Adversarial Attacks on Computer Vision Models
A cybersecurity project on the fragility of computer vision models. I trained a YOLOv8 model, then crafted pixel-level perturbations to fool it — demonstrating how small, targeted changes break model predictions.
[Repo](https://github.com/Abdelmalek05/pixel-adversarial-attack) · [Project Site](https://pixel-adversarial-attack-website.vercel.app/)
`Python` `YOLOv8` `Computer Vision` `Adversarial ML`

### 🩺 Doctor's Cabinet Management System
A desktop application for medical practices: patient and consultation management, plus an AI assistant powered by Gemini that builds context from patient records and consultation history to support clinical decisions.
_Demo link coming soon_
`Electron` `Next.js` `SQLite` `Gemini API`

### 📱 CRM Mobile App
A cross-platform CRM with a Flutter front end and a containerized Django REST backend (JWT auth, PostgreSQL).
[Backend Repo](https://github.com/Abdelmalek05/crm_app_django_server) · [Demo Video](https://drive.google.com/file/d/1-egNVt45YlXpq1FWX_MBX_BFEfrUaMYq/view)
`Flutter` `Django` `PostgreSQL` `Docker`

### 📄 AI Resume Analyzer
A web app that parses and scores resumes, surfacing actionable feedback for candidates. The analysis is powered by Claude 3.5 Sonnet.
[Repo](https://github.com/Abdelmalek05/ai-resume-analyzer) · [Live App](https://jsm-ai-resume-analyzer-82-ls5u8.puter.site/)
`React` `LLM API` `Web`

### 🤖 Mini CNC Drawing Machine
Built with GDG ENSIA — a small CNC machine driven by Arduino and Raspberry Pi that takes a vector image as input and physically draws it. A hands-on dive into automation and embedded systems.
`Arduino` `Raspberry Pi` `Embedded`
